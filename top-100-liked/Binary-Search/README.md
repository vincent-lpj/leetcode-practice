# Binary Search 上手技巧

## 核心模板

先把这个框架背熟，后面所有题都是在它上面改：

```python
left, right = 0, len(nums) - 1

while left <= right:
    mid = left + (right - left) // 2   # 避免 (left+right) 大数溢出（Python 无所谓，但是好习惯）
    if nums[mid] == target:
        return mid
    elif nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1

return -1
```

---

## 三个最容易出错的细节

| 问题                         | 结论                                                                                              |
| ---------------------------- | ------------------------------------------------------------------------------------------------- |
| 循环条件用 `<` 还是 `<=`     | 区间是否包含右端点：`[left, right]` 闭区间用 `<=`，`[left, right)` 半开区间用 `<`                 |
| `mid` 往哪边收缩             | 找到目标时直接返回就用 `left = mid + 1` / `right = mid - 1`；找边界时需要继续收缩，不能跳过 `mid` |
| 返回值是 `left` 还是 `right` | 搜索结束后 `left == right + 1`，找"插入位置"或"左边界"返回 `left`，找"右边界"返回 `right`         |

---

## 遇到变种题时的判断思路

1. **数组是否有序？** 二分的前提是单调性，不一定是严格排序，但某一侧必须满足某个性质。
2. **找的是"某个值"还是"某个位置"？** 找值直接比较；找边界（第一个/最后一个）要在找到后继续收缩。
3. **能否把问题转化为"判断函数"？** 即 `check(mid)` 返回 True/False，让左半段全为 False、右半段全为 True（或反过来），然后二分找边界。这是通用技巧，适用于 Hard 题。
