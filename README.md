# LeetCode Practice

A personal collection of LeetCode solutions written in Python3, using the [LeetCode VS Code extension](https://marketplace.visualstudio.com/items?itemName=LeetCode.vscode-leetcode).

## Structure

Each file is named `<problem-id>.<problem-title>.py` and contains the solution class generated by the extension.

## Solutions

| #   | Title                                       | Difficulty | Approach                                                        |
| --- | ------------------------------------------- | ---------- | --------------------------------------------------------------- |
| 1   | [Two Sum](1.two-sum.py)                     | Easy       | Hash map — one-pass, O(n) time / O(n) space                     |
| 9   | [Palindrome Number](9.palindrome-number.py) | Easy       | Reverse half of the number mathematically, no string conversion |

## Notes

- All solutions are in **Python3**.
- Comments in source files explain the reasoning behind each approach.
